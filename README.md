# Gemini Folder Extension
厭倦 Gemini 對話散落著嗎？ Gemini Folder Extension 為你帶來簡單卻強大的資料夾管理方案！
套件連結: https://chromewebstore.google.com/detail/gemini-folder-extension/lgebiepfcgjdgonlgigljommlkmddali


## 🚀 核心功能
* 建立多個資料夾，有系統地分類你的對話
* 直覺的拖放操作，快速將對話分類
* 自定義資料夾名稱和顏色，讓你一目了然
* 自適應 light/dark mode，當你切換 theme，插件的顯示也會跟著變更
* 覆蓋式設計，讓你擁有原始操作和插件功能
* 自動同步，只要你的瀏覽器開啟自動同步，那你可以在各處同步你的資料夾


## 💡 隱私
* 你的資料只會存在你的帳號上，如果你沒有開啟同步則只存在 local 端，如果開啟同步，則在你的帳號各地同步資料
* 該插件保證不蒐集你的對話內容，僅讀取對話名稱


## ❗ 注意
* 因為同步資料有大小上限，所以對話的一些屬性(例如名稱) 將存在 local 端，當你在別台電腦開啟時，會看到對話紀錄變成 "Unknown" 這是正常的，只需要滾動一下之前的對話紀錄，讓他重新載入即可
* 此專案將不積極維護，在 Gemini 原生支援資料夾功能後廢棄
* 功能許願 or 遇到問題？ 歡迎開 [issue](https://github.com/mark0613/gemini-folder-extension/issues)
